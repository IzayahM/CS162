This application uses most of the code we made in cs162. 
To use it you have to start by pressing the start button. 
Then you have access to the pause and next button. 
To use the next button you need to press it and then start to change the intervals. 
Once time is up you need to press the next button then the start button. 
There is a widget below the pause button, It is used to reflect or create goals for the study periods.
The text can be modified of deleted.
